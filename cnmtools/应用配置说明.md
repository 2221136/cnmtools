## 🚀 如何运行

### 运行应用
```bash
# Android
flutter run -d android

# iOS
flutter run -d ios

# Web
flutter run -d chrome

# Windows
flutter run -d windows

# macOS
flutter run -d macos

# Linux
flutter run -d linux
```

### 构建应用
```bash
# Android APK
flutter build apk

# Android App Bundle
flutter build appbundle

# iOS
flutter build ios

# Web
flutter build web

# Windows
flutter build windows

# macOS
flutter build macos

# Linux
flutter build linux
```

## 📝 注意事项

1. 如果需要更换图标，只需替换根目录的 `logo.png` 文件，然后运行：
   ```bash
   flutter pub run flutter_launcher_icons
   ```

2. Windows平台的应用标题显示为可执行文件名（cnmtools），如需修改需要在 `windows/CMakeLists.txt` 中修改 `BINARY_NAME`

3. 所有平台的图标已自动生成，无需手动配置
